# Скриншоты для сдачи

Сюда положите два изображения (после локального запуска проекта):

| Файл | Что снять |
|------|-----------|
| `main-form.png` | Страница с загруженными вопросами и полями ввода (до отправки). |
| `thank-you.png` | Экран с сообщением **«Спасибо!»** после успешной отправки формы. |

**Как сделать:** запустите backend, откройте frontend (через локальный сервер или как указано в корневом README), сделайте скриншоты и сохраните под именами выше.

В корневом `README.md` есть раздел «Скриншоты» — при необходимости вставьте туда превью или ссылки на эти файлы.
