ЗАДАНИЕ
Нужно реализовать простое full-stack приложение «Мини-анкета», состоящее из backend и frontend.

Функциональность:
1. Backend API:
- GET /questions — возвращает список вопросов анкеты, жёстко заданных, 3–5 штук.
- POST /answers — принимает ответы пользователя и сохраняет их в памяти приложения (например, list/array/map, без БД).

2. Frontend:
- загружает вопросы с backend;
- отображает их в простом HTML/JS интерфейсе;
- отправляет заполненные ответы через POST /answers;
- после успешной отправки показывает сообщение «Спасибо!».

Технологии:
- Backend: Python.
- Frontend: чистый HTML + JS.
Сделай самый простой, понятный и легко запускаемый вариант.
