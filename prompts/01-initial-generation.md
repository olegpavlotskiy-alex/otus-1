# Промпт: начальная генерация проекта «Мини-анкета»

Текст задания (полный запрос на реализацию full-stack приложения):

- Backend: Python 3, Flask, GET `/questions`, POST `/answers`, хранение ответов в памяти, без БД.
- Frontend: чистый HTML/CSS/JS, загрузка вопросов, форма, POST ответов, сообщение «Спасибо!», обработка ошибок.
- Структура: `backend/`, `frontend/`, `prompts/`, `screenshots/`, `README.md`, `.gitignore`.
- README: описание, технологии, запуск, API, промпты, скриншоты.
- CORS при необходимости для локального запуска.
- Без Docker, без React/Vue/TS, без лишней архитектуры.

Формат ответа ассистента: план → структура → код по файлам → команды → проверка → README → промпты → чеклист → git.
